# **App Name**: eSports HQ

## Core Features:

- User Authentication: Allow players to create accounts and set their usernames, region, and profile picture using Google or email/password authentication.
- Tournament Creation (Admin): Enable admins to add tournament details such as name, date, entry fee, and QR code image.
- Tournament Listing: Display a list of active and open tournaments for players to browse. Uses Firestore.
- Team Registration: Allow team leaders to register their teams by entering team name, a list of 4-6 player usernames, and uploading a payment screenshot. Uses Firestore.
- Admin Approval: Enable admins to view pending team registrations and approve or reject them. Uses Firestore.
- Player Dashboard: Display the tournaments a player is registered in, along with the registration status and team details. Uses Firestore.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey a sense of professionalism and focus.
- Background color: Light gray (#E0E0E0) for a clean and modern look.
- Accent color: Electric Green (#7CFC00) to highlight important actions and provide contrast.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and neutral look.
- Use sharp, vector-based icons to represent different elements of the website, such as tournaments, teams, and players.
- Design a clean, organized layout that is easy to navigate. Use clear sections and visual cues to guide users through the different pages.
- Incorporate subtle animations and transitions to enhance the user experience and provide visual feedback.
